# HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure of a webpage, which can be enhanced and modified by other technologies like CSS and JavaScript.

## Basic HTML Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>

This is the basic structure of an HTML document, which includes the doctype, html, head, and body elements.
