# Basics of Machine Learning

Machine Learning (ML) is a subset of AI that focuses on the development of algorithms that can learn from and make predictions on data.

## Supervised Learning

In supervised learning, the model is trained on labeled data. The goal is to learn a mapping from inputs to outputs.

## Unsupervised Learning

In unsupervised learning, the model is trained on unlabeled data. The goal is to find hidden patterns or intrinsic structures in the input data.
